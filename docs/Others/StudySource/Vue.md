#### 优秀回答
[网上都说操作真实 DOM 慢，但测试结果却比 React 更快，为什么？](https://www.zhihu.com/question/31809713)
尤大观点：
1\. 框架保证的是在不需要手动优化的情况下依然可以提供过得去的性能。
2\. Virtual DOM render + diff 确保无论数据变化多少，每次重绘的性能都可接受的。
