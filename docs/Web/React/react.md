## 优势
- 避免DOM-API繁琐操作
- 避免直接操作DOM引发的重绘重拍
- 引入组件化（包括html、css、图片、字体、音视频，对比模块化的仅js拆分）编码方案，声明式编程，代码复用率高
- React native可以使用React语法进行移动端开发
- 使用虚拟DOM + 优秀Diffing算法，尽量减少与真实DOM的交互
